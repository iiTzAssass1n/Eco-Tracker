# Eco-Tracker
Eco-Tracker nasce da un'idea semplice: e se guadagnassimo punti ogni volta che facciamo qualcosa di buono per il pianeta? L'app permette agli studenti di registrare le proprie azioni sostenibili quotidiane — usare la borraccia, andare a scuola in bici, fare la raccolta differenziata — e di trasformarle in un punteggio che cresce giorno dopo giorno.
